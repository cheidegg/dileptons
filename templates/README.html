This readme briefly gives an overview on the different outputs produced by the macro. Grouped in different folders in different levels these outputs depict different quantities, or the same quantities but with different cuts applied or datasets used.

LEVE1 - Cut Definition (*)
(1) ETH - 'normal' ETH set of cuts
(2) UFL - ETH set of cuts, additionally also btag cut (0.679) and delta-phi cut (2.0) on away-jet according to UFL

LEVEL 2 - Jet Pt Correction (*)
(1) corrected_jet - using corrected JetPt, rest unchanged
(2) corrected_jetd0 - changed DO cut (50 microns) and using corrected JetPt, rest unchanged
(3) corrected_tot - changed D0 cut (50 microns), relative ISO (5%) and using corrected JetPt
(4) uncorrected - 'normal' D0 cut (100 microns), relative ISO (10%) and using JetRawPt

LEVEL 3 - Pile Up Reweighting (*)
(1) PUweight - using PUWeight from total run applied to run D only [DON'T USE]
(2) PUweight_complete - using PUWeight from total run applied to total run
(3) PUweight_runD - using PUWeight from run D applied to run D only
(4) noPUweight - no PUWeight used at all for run D
(5) runOld - ''old'' dataset from run D used [DON'T USE]
(6) runOldB - ''old'' datasets from run B used [DON'T USE]

LEVEL 4 - High-Level Trigger (*)
(1) Ele8J - using HLT ELE8_JET30_TIGHT on data only [ELECTRONS]
(2) Ele8JMC - using HLT ELE17_JET30_TIGHT on data and MC [ELECTRONS]
(3) Ele17J - using HLT ELE17_JET30_TIGHT on data only [ELECTRONS]
(4) Ele17JMC - using HLT ELE17_JET30_TIGHT on data and MC [ELECTRONS]
(5) Mu17 - using HLT Mu17 on data only, current version (including jet-lepton cleaning, veto cut) [MUONS]
(6) Mu17_jec - using HLT Mu17 on data only, including jet-lepton cleaning, veto cut, jet-energy correction [MUONS]
(7) Mu17_jlc - using HLT Mu17 on data only, including jet-lepton cleaning [MUONS]
(8) Mu17_nojlcvetojec - using HLT Mu17 on data only [MUONS]
(9) Mu17_veto - using HLT Mu17 on data only, including jet-lepton cleaning and veto cut [MUONS]
(10) Mu17MC - using HLT Mu17 on data and MC [MUONS]
(11) Mu40 - using HLT Mu40 on data only [MUONS]
(12) Mu40MC - using HLT Mu40 on data and MC [MUONS]
(13) none - no Trigger used (mainly testing) [ELECTRONS]

LEVEL 5 - Scaling (+)
(1) fit_weighted - scaling QCD + EWK to data via RooFit
(2) qcd_weighted - scaling QCD to data
(3) qcdwjets_weighted - scaling QCD + WJets to data
(4) unweighted - no scaling at all

LEVEL 6 - Modules (+)
(1) adhoc - some hard-coded ad hoc stuff, testing, etc.
(2) compare - all normalized comparison plots, i.e. comparing one quantity for two datasets (normalized)
(3) fakerates_1d - all fake rate plots in one variable, also projections from 2dmap
(4) fakerates_2d - all fake rate plots in two variables (2dmaps, e.g. JetPt differences)
(5) fakerates_2dct - all fake rate plots in two variables (2dmaps) for closure test
(6) plots_1d - all 'normal' plots in one variable
(7) plots_2d - all 'normal' plots in two variables
(8) zoom_jpt - 1d projections for relative and absolute differences in JetPt
(9) zoom_met - 1d projections for MET


(*)... created by c++ Fakerates script
